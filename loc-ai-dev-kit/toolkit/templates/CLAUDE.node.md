# Node Project Instructions

## Development Commands
- Install dependencies: `npm install`
- Run tests: `npm test`
- Run build: `npm run build`
- Run lint: `npm run lint`
- Run dev server: `npm run dev`

## Coding Guidelines
- Prefer existing package scripts and framework conventions.
- Validate external input at API boundaries.
- Avoid adding dependencies unless they clearly reduce complexity.
- Keep frontend behavior verified in a browser when UI changes are made.

## Claude Workflow
- Inspect package scripts before running commands.
- Run targeted tests or type checks after code changes.
